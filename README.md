# 📱 Mobile Price Prediction Web App

A **Machine Learning powered web application** that predicts the estimated price of a smartphone based on its technical specifications.

This project demonstrates the **complete machine learning workflow** — from data analysis and model building to real-world deployment using **Streamlit**.

# 📌 Project Overview

Smartphone prices depend on multiple hardware and performance factors such as:

* RAM
* Internal Storage
* Camera Quality
* Battery Capacity
* Processor Specifications
* Display Resolution

This project uses a **Machine Learning regression model** trained on smartphone specifications to estimate the expected price of a mobile device.

Users can enter smartphone specifications through an **interactive web interface** and instantly receive a **predicted price**.

---

# 🧠 Machine Learning Workflow

This project follows a standard **Machine Learning pipeline**:

1. Data Collection and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Selection
4. Model Training using **Linear Regression**
5. Model Evaluation
6. Model Serialization using **Pickle**
7. Web App Deployment using **Streamlit**

---

# 📊 Features Used for Prediction

The model predicts mobile price using the following specifications:

* Market Popularity Score
* Device Weight
* Display Resolution
* Pixel Density (PPI)
* CPU Cores
* CPU Frequency
* Internal Storage
* RAM
* Rear Camera Megapixels
* Front Camera Megapixels
* Battery Capacity
* Phone Thickness
