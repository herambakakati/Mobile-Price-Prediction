📱 Mobile Price Prediction Web App
A machine learning powered web application that predicts the estimated price of a smartphone based on its technical specifications. This project demonstrates the complete workflow from data analysis and model building to real-world deployment using Streamlit.

Developed by Heramba Kakati

🚀 Live Demo
Access the deployed application here:

https://mobile-price-prediction-app-02.streamlit.app/

📌 Project Overview
Smartphone prices depend on multiple hardware and performance factors such as RAM, storage, camera quality, battery capacity, and processor specifications.

This project uses a Machine Learning model trained on smartphone specifications to estimate the expected price of a mobile device.

Users can enter the phone specifications through an interactive web interface and instantly receive a predicted price.

🧠 Machine Learning Workflow
The project follows a standard ML pipeline:

Data collection and preprocessing
Exploratory Data Analysis (EDA)
Feature selection
Model training using Linear Regression
Model evaluation
Model serialization using Pickle
Deployment using Streamlit
📊 Features Used for Prediction
The model uses the following smartphone specifications:

Market popularity score
Device weight
Display resolution
Pixel density (PPI)
CPU cores
CPU frequency
Internal storage
RAM
Rear camera megapixels
Front camera megapixels
Battery capacity
Phone thickness
